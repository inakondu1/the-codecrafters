## SENTINEL REBORN
This Program was build from the scratch by the struct

It is a text processing tool which reads text from file, format and transform a text base on command in the input. 

The struct Used some functions like 
1. fixQuote: to wrape the quote round a specific word or sentence.
2. fixArticle:  it is grammer sensitive, it changes a to an when the next first letter start with a vowel
3. hexToDecimal: it works based on command like "(hex)", it convert the previous word before "(hex)" to Decimal number
4. Punctuation: it draw the punctuation close to the word before it and give space to the word after it 
5. binToDecimal: it work based on a given marker such as (bin)
6. fixCommand: it handles all case of uppercase, capitalize and lowercase. it uses the following commands ("up"), ("cap"), ("low"), ("up, 2"), ("cap,3"), ("low, 4")
