# Go Variables

In Go, they are different types of variables.

### Integrer  :
 the short form for this is "int" it is a representation of whole numbers. e.g 123

### float:  
 The float has different numbers eg float 32, 64 etc. it is a number with decimals, such as 77.77 or -69.75


### string :
 These  stores text, such as "Hello World!" . for a word to be identify as a string, it must carry the double quote "" 
 
### bool:
these  stores data  with two states: true or false.

### Go Variable Naming Rules
In naming your variables, they are some rules guiding it, for example, you cant name a variable starting with a number or point. Also, variables naming are case sensitive expecially in spelling. 
a varable naming has no  limite of words but it shouldnt have space inbetween the word.
## They are different ways of naming a varables


### Camel Case

In this camel to case, each word except the first, starts with a capital letter.

### Pascal Case

Each word starts with a capital letter

### Snake Case

Each word is separated by an underscore character.
